# Gradient-Trader
A end to end   trading bot for cryptocurrency.


## How to run
```javascript

git clone https://github.com/viditkhemka63/Gradient-Trader.git
cd Gradient-Trader 

npm install --only=production

cd exchange
npm install --only=production
cd ..

cd gekko/web/vue
npm install

# path to webserver
cd gekko/web
# launch the server - we use this API
node server

# path to vue app
cd vue
npm run serve

```
